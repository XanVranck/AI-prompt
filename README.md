## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/XanVranck/AI-prompt
    ```

2. Navigate to the project directory:

    ```bash
    cd AI-prompt
    ```

3. Install the dependencies:

    ```bash
    npm install
    ```

## Usage

```bash
node index.js
```

Go to localhost:3000
